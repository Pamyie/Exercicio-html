solilóquio
No drama, onde um personagem fala a si mesmo, representando seus pensamentos ou sentimentos internos e no processo, transmitindo-os ao público (mas não a outros personagens).
monólogo
No drama, onde um personagem fala seus pensamentos em voz alta para compartilhá-los com o público e com outros personagens presentes.
aparte
No drama, onde um personagem compartilha um comentário apenas com o público para efeito humorístico ou dramático. Isso geralmente é um sentimento, pensamento ou parte de informação de fundo adicional.

Representando código de computador

O elemento <code> é usado para representar código de computador. Ele é geralmente renderizado em uma fonte monoespaçada para indicar que é código.

O elemento <pre> é usado para representar texto pré-formatado, onde os espaços em branco e quebras de linha são preservados. Ele é útil para exibir código de computador ou qualquer outro texto onde a formatação é importante.

O elemento <samp> é usado para representar a saída de um programa de computador. Ele é geralmente renderizado em uma fonte monoespaçada para indicar que é saída do programa.

O elemento <kbd> é usado para representar a entrada do usuário, como pressionar uma tecla ou clicar em um botão.

O elemento <var> é usado para representar uma variável em uma expressão matemática ou científica.

Marcando hora e data

O elemento <time> é usado para marcar uma hora ou data. Ele pode ser usado para marcar uma data específica, como um aniversário, ou uma hora específica, como um horário de reunião.
